## Best Sources for Vendor-to-Customer DC Deployment Data

**Equinix CIF (Ashburn DC15)** is the single most important multi-vendor showcase — ZutaCore, Accelsius/JetCool/Flex all present there. Check Equinix press releases for new CIF partners.

**Vendor press release pages (primary source):**
- ZutaCore: blog.zutacore.com/press-releases + zutacore.com/customer-stories (40+ named deployments)
- Accelsius: accelsius.com (DarkNX, iM Data Centers, Equinix CIF, Park Place)
- JetCool/Flex: jetcool.com/post/ (Sabey, Start Campus, Telehouse, Equinix CIF)
- Motivair: motivaircorp.com/news/ (3 US exascale supercomputers confirmed)
- Asperitas: asperitas.com/news (maincubes, Shell, Credit Agricole, UNICOM)
- Iceotope: iceotope.com/company/news (EcoDataCenter Falun is primary confirmed site)

**Component suppliers (Boyd/Eaton, CoolIT/Ecolab)** do NOT disclose customer DC sites — they sell cold plates/CDUs to server OEMs. 5M+ cold plates shipped (Boyd), 300+ DCs (CoolIT) but no facility names public.

**Industry publications for deployment announcements:**
- DatacenterDynamics (DCD): best for breaking deployment news
- Data Center Frontier: good for detailed case studies
- HPCwire: best for HPC/supercomputer cooling vendor confirmations
- BusinessWire/PR Newswire: official announcements from vendors

**National labs (LLNL, ORNL, ANL, Sandia, LANL, NREL)** are the most thoroughly documented liquid cooling customers — useful as reference benchmarks.

**Telehouse South (London)** is a secondary multi-vendor showcase (JetCool, Accelsius, EkkoSense, Legrand) — launched early 2025.
